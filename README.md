# 梦处方 · AI 睡眠助手

> AI 原生睡眠健康微信小程序，构建“监测→分析→干预→反馈”的完整闭环。

##项目简介

梦处方是一个面向 C 端的 AI 睡眠健康微信小程序，旨在通过 AI 技术将睡眠数据转化为可执行的个性化健康行动方案，解决传统助眠产品“只测不管”的痛点。

## 核心功能亮点

- **AI 织梦**：根据用户情绪状态，实时生成个性化的助眠睡前故事。
- **AI 环境监测**：通过光线与噪音识别，智能评估入睡环境并提供改善建议。
- **AI 睡眠问答**：基于 RAG 技术，提供专业、安全的睡眠知识问答。
- **完整 AI 闭环**：实现从“数据采集 → 深度分析 → 病因诊断 → 干预推荐”的端到端体验。

## 技术能力

本项目涉及的核心 AI 能力包括：
- **大语言模型 (LLM)**：用于 AI 睡眠助手的多轮对话与文本生成。
- **检索增强生成 (RAG)**：确保睡眠知识问答的专业性与可信度。
- **计算机视觉 (CV)**：用于睡眠环境的光线与噪音评估。
- **时序数据分析**：用于睡眠质量评分与病因诊断。
